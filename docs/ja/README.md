# CronWorkflow Replicator ドキュメント

CronWorkflow Replicator のドキュメントへようこそ。このツールは、異なる値を使用してKubernetes CronWorkflowを複製するのに役立ちます。

## 目次

- [使用方法](usage.md) - ツールの使用方法（Docker使用を含む）
- [設定](configuration.md) - 設定オプションとパス解決について

## 概要

CronWorkflow Replicator は以下のことができるシンプルなツールです：

- 単一の設定から複数のCronWorkflowマニフェストを生成
- 生成された各マニフェストに異なる値を適用
- 一貫した構造のためのベースマニフェストテンプレートを使用
- ポータビリティのために設定ファイルに対する相対パスを解決

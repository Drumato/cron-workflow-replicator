# CronWorkflow Replicator Documentation

Welcome to the CronWorkflow Replicator documentation. This tool helps you replicate Kubernetes CronWorkflows with different values.

## Table of Contents

- [Usage Guide](usage.md) - How to use the tool, including Docker usage
- [Configuration](configuration.md) - Configuration options and path resolution

## Overview

CronWorkflow Replicator is a simple tool that allows you to:

- Generate multiple CronWorkflow manifests from a single configuration
- Apply different values to each generated manifest
- Use base manifest templates for consistent structure
- Resolve paths relative to configuration files for portability
